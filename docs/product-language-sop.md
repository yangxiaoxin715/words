# 产品语言 SOP

## 1. 名称和状态语言要同一套

- 产品名：和单词交朋友。
- 状态语言：新朋友、有点眼熟、老朋友。
- 行为语言：见面、陪伴、交朋友、再见一面、再来一轮。

判断标准：用户看到标题、按钮、数据和结果时，要感觉是在和同一套产品语言对话。

## 2. 避免偏对抗的词

- 避免：猎、捕获、瞄准、消灭、攻克。
- 优先：见面、交朋友、眼熟、老朋友、新朋友。

自动模式内部逻辑可以仍然是快速识别，但用户界面不强调对抗感。

## 3. 首页固定文案

首页面向手机使用，语言要轻，但不要改成探险、闯关或吉祥物风。

- 顶部标题区：显示 `Welcome to make friends with words!`，不拆成词块，不写说明文字。
- 顶部右侧：编号放在“退出”按钮左边，只显示编号本身，不加“学习编号”“词友号”“交友名片”等标签。
- 手机顶部布局：左侧标题区必须可收缩，右侧编号和退出按钮必须完整露出；英文信号要完整显示 `words!`，不能用省略号截断。
- 四个数据卡片：今日见面、今天陪伴、累计见面、连续来玩。
- 阶段按钮：召唤剩余 N 个、再见一面。
- 模式按钮：自动模式、手动模式。
- 主按钮：开始交朋友。
- 英语信号：只出现一次，放在顶部标题区。

判断标准：孩子或家长打开首页，第一眼能知道这是英语单词工具，第二眼能知道下一步点哪里。
